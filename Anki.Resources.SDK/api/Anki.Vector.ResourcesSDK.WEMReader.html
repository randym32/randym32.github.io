    Class WEMReader

    This is used to open up the WEM segment in the BNK file and setup something that NAudio can access.

    public class WEMReader : IDisposable
    Remarks

    A lot of the format info came from:

    • http://bobdoleowndu.github.io/mgsv/documentation/soundswapping.html
    • https://github.com/losnoco/vgmstream/blob/master/src/meta/wwise.c
    Examples

    At the moment the format within the WEM files is not well-known or understood. There isn't yet software that we can use to decode on our own. (There are separate stand-alone programs, and the VGMStream library may be able to decode the files, if someone can link it in.)

    The hope is that some day there will be a decoder and that you can pipe the data into NAudio using code like:

        var WEM = new WEMReader("some path to a .wem");
    WEM.Open();
    using var blockAlignedStream = new BlockAlignReductionStream(
       new RawSourceWaveStream(WEM.DataStream(),
       new WaveFormat((int)WEM.sampleRate, (int)WEM.bitsPerSample, (int)WEM.numChannels)));
    var waveOut = new WaveOut();
    waveOut.Init(blockAlignedStream);
    waveOut.Play();
