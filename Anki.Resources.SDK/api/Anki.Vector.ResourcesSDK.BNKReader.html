    Remarks

    The sound bank files decoding is based on the format documented at http://wiki.xentax.com/index.php/Wwise_SoundBank_(*.bnk)

    Vector's Bank files have the following sections in them:

    • BKHD
    • INIT
    • STMG
    • HIRC
    • ENVS
    • PLAT
    • DIDX
    • DATA
    • STID - this is used once, just to call out the Vector_VO .bnk file

    The following may be added in the future

    • Switch groups
    • Switches
    • State group
    • States
    • Custom states
    • Game parameters
    • Audio buses
